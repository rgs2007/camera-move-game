Experimental Input Game

Camera Move Game

A small prototype for using live camera motion as a game controller. The first version uses frame-to-frame image difference to estimate movement direction and steer a ball toward a goal.

How It Works

Move your body left, right, up, or down in front of the camera. The game tracks where motion is happening in the video and uses that moving center point to push the ball across the playfield.

If the browser blocks the camera, try the same page on http://localhost:4173 or reset site camera permissions.

The playfield also supports arrow keys or WASD, so game movement can still be tested even when camera access is unavailable.
